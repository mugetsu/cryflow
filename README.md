# cryflow

> (/ˌcryptˈflow/) Show off words in a cryptic manner.

## Project setup
```
yarn install
```

### Compiles and minifies for production
```
yarn build
```

### Usage
```html
<cryflow
  :phrases="[
    'Show off',
    'words in a',
    'cryptic manner'
  ]"
/>
```
