> (/ˌcryptˈflow/) Show off words in a cryptic manner.

### Usage
```vue
<template>
  <cryflow
    :phrases="[
      'I am',
      'Randell',
      'a developer',
      'based in Philippines'
     ]"
   />
</template>

<script>
import Cryflow from 'cryflow'

export default {
  components: {
    Cryflow
  }
}
</script>
```
